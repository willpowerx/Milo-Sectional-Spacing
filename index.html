Static Variants

Fixed padding-block values that do not change across breakpoints. Each .spacing-*-static class maps directly to a primitive spacing or layout token — use when a layout needs the same space at desktop, tablet, and mobile.

.spacing-*-static padding-block — fixed value, all breakpoints
.spacing-4xs-static 8px — --s2a-spacing-xs
.spacing-3xs-static 16px — --s2a-spacing-md
.spacing-2xs-static 24px — --s2a-spacing-lg
